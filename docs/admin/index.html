How It Works

  5. Your live site updates in 1-2 minutes

Change Admin Password

The default password hash is in docs/admin/app.js. To change it:

  1. Generate SHA-256 hash of your new password (use any online tool)
  2. Replace ADMIN_PASSWORD_HASH in docs/admin/app.js
  3. Commit and push the change
